# Advent of Code 2025

To install dependencies:

```fish
bun install
```

Save AOC_COOKIE as environment variable

```fish
set -U AOC_COOKIE <cookie>
```

To run:

```fish
./download.fish <year> <day>
bun run <day>.ts
```

This project was created using `bun init` in bun v1.2.23. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
